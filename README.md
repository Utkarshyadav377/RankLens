# RankLens

**See your Codeforces profile clearly.**

Topic ratings, weak spots, and learning paths — built for competitive programmers.

Built by [@Utkarshyadav377](https://github.com/Utkarshyadav377).

## Features

- **Topic Ratings** — Elo-style estimates across DP, graphs, greedy, and more
- **Gap Finder** — Spot where your solve rate drops
- **Deep Analytics** — Rating histograms and per-problem stats
- **Skill Map** — Interactive prerequisite graphs

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and enter a Codeforces handle.

## Deploy to Vercel

1. Push to [github.com/Utkarshyadav377/cf-coach](https://github.com/Utkarshyadav377/cf-coach)
2. Import the repo at [vercel.com/new](https://vercel.com/new)
3. Deploy — Vite is auto-detected

## Tech Stack

React · Vite · Tailwind CSS · Framer Motion · Recharts · React Flow
